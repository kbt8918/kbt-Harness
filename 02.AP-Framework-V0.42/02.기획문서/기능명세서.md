## 기능 상세

### F-001. 실시간 위치 수집·전송

| 항목 | 내용 |
|------|------|
| 설명 | 부모님 단말의 GPS 위치를 주기적으로 수집해 서버로 전송한다 |
| 입력 | 단말 Geolocation 좌표(위도/경도/정확도/타임스탬프), 위치 권한 동의 |
| 처리 | 위치 권한 확인 → 주기(OI-01 확정값)마다 좌표 수집 → 인증 토큰과 함께 위치 전송 API 호출 → 서버 저장/브로드캐스트 |
| 출력 | 서버 저장 성공 응답, 가족 화면 실시간 위치 갱신 트리거 |
| 예외 처리 | 위치 권한 거부 시 안내·재요청 / GPS 미수신 시 마지막 위치 유지 + "위치 확인 불가" 표시 / 네트워크 끊김 시 재전송 큐 |
| 관련 화면 | 부모님 화면 |
| 관련 API | `POST /api/location` (위치 전송), 위치 브로드캐스트(SSE/Realtime) |

### F-002. 긴급 알림 발송 (1-터치)

| 항목 | 내용 |
|------|------|
| 설명 | 부모님이 큰 긴급 버튼을 1회 터치하면 매핑된 가족에게 즉시 알림을 발송한다 |
| 입력 | 긴급 버튼 터치 이벤트, 부모님 계정·현재 위치 |
| 처리 | 터치 1회 감지 → (오작동 방지 최소 처리) → 긴급 알림 API 호출 → 매핑 가족 대상 실시간 알림 브로드캐스트 + 현재 위치 첨부 |
| 출력 | 가족 화면 긴급 알림 강조 표시(F-005), 부모님 화면 발송 피드백(REQ-P-003) |
| 예외 처리 | 네트워크 실패 시 재시도 + 실패 시 "전송 실패, 다시 시도" 표시 / 중복 연타 시 짧은 시간 내 1건으로 처리(디바운스) / 매핑 가족 없음 시 관리자 통보 |
| 관련 화면 | 부모님 화면 → 가족 화면 |
| 관련 API | `POST /api/alert/emergency`, 알림 브로드캐스트(SSE/Realtime) |

### F-003. 부모님 화면 고령자 UI

| 항목 | 내용 |
|------|------|
| 설명 | 큰 버튼·큰 글씨·고대비의 단순 화면과 "안심·연결" 톤 메시징을 제공한다 |
| 입력 | 부모님 로그인 세션 |
| 처리 | 부모님 역할 확인 → 단일 핵심 버튼(긴급) 중심 레이아웃 렌더 → 발송 상태 시각 표시 영역 구성 |
| 출력 | 초대형 긴급 버튼, 큰 글씨 상태 텍스트, 감시 연상 배제 문구 |
| 예외 처리 | 세션 만료 시 단순 재로그인 안내 / 저사양 단말에서도 핵심 버튼 우선 렌더 |
| 관련 화면 | 부모님 화면 |
| 관련 API | (인증 F-011), 화면 설계 #9 / 디자인가이드 #13 연계 |

### F-004. 부모님 위치 지도 표시

| 항목 | 내용 |
|------|------|
| 설명 | 매핑된 부모님의 현재 위치를 지도에 마커로 표시한다 |
| 입력 | 가족 로그인 세션, 매핑된 부모님 식별자, 부모님 최신 위치 |
| 처리 | 권한·매핑 확인 → 지도 API 초기화 → 부모님 최신 좌표 마커 표시 → 위치 갱신 시 마커 이동 |
| 출력 | 지도 + 부모님 위치 마커, 최종 갱신 시각 |
| 예외 처리 | 위치 데이터 없음 시 "위치 확인 불가" / 지도 API 로드 실패 시 대체 안내 / 매핑 부모님 없음 시 빈 상태 안내 |
| 관련 화면 | 가족 화면 |
| 관련 API | `GET /api/location/:parentId`, 지도 API(카카오/네이버/Google) |

### F-005. 실시간 위치·알림 수신

| 항목 | 내용 |
|------|------|
| 설명 | 위치 갱신과 긴급 알림을 수 초 내 가족 화면에 반영하고, 긴급 알림은 강조 표시한다 |
| 입력 | 실시간 채널 구독(가족 세션 기준), 위치/알림 이벤트 |
| 처리 | SSE/WebSocket/Realtime 구독 → 위치 이벤트 수신 시 마커 갱신 → 긴급 알림 이벤트 수신 시 일반 알림과 구분되는 강조 UI 표시 |
| 출력 | 실시간 위치 마커 갱신, 긴급 알림 강조 배너/사운드 |
| 예외 처리 | 연결 끊김 시 자동 재연결 / 재연결 중 누락 이벤트 폴링 보정 / 권한 없는 채널 구독 차단 |
| 관련 화면 | 가족 화면 |
| 관련 API | 실시간 채널(SSE `/api/stream` 또는 Supabase Realtime) |

### F-006. 바로 전화 연결

| 항목 | 내용 |
|------|------|
| 설명 | 가족 화면에서 버튼 클릭 시 부모님에게 즉시 통화 연결한다 |
| 입력 | 가족 클릭 이벤트, 매핑된 부모님 전화번호 |
| 처리 | 매핑·권한 확인 → 부모님 전화번호 조회 → `tel:` 링크 실행 |
| 출력 | 단말 통화 앱 실행(부모님 번호로 발신) |
| 예외 처리 | 전화번호 미등록 시 버튼 비활성 + 안내 / tel: 미지원 환경 시 번호 표시 |
| 관련 화면 | 가족 화면 |
| 관련 API | `GET /api/member/:parentId/phone` (권한 검증 포함) |

### F-007. 가족 채팅

| 항목 | 내용 |
|------|------|
| 설명 | 가족 단위 실시간 채팅방에서 메시지를 송수신한다 |
| 입력 | 가족 세션, 채팅방(가족 그룹) 식별자, 메시지 텍스트 |
| 처리 | 채팅방 권한 확인 → 메시지 저장 → 실시간 채널로 같은 가족 구성원에게 전파 → 이력 로드(페이지네이션) |
| 출력 | 채팅방 메시지 목록 실시간 갱신 |
| 예외 처리 | 빈 메시지 차단 / 전송 실패 시 재시도 표시 / 권한 없는 방 접근 차단 |
| 관련 화면 | 가족 화면 |
| 관련 API | `GET/POST /api/chat/:roomId/messages`, 채팅 실시간 채널 |

### F-008. 회원 조회

| 항목 | 내용 |
|------|------|
| 설명 | 관리자가 연동 회원(부모-가족 매핑) 정보를 조회한다 |
| 입력 | 관리자 세션, 검색/필터 조건 |
| 처리 | 관리자 권한 확인 → 회원·매핑 데이터 조회 → 목록/상세 반환(페이지네이션) |
| 출력 | 회원 목록, 부모-가족 매핑 관계 상세 |
| 예외 처리 | 권한 없음 시 차단 / 결과 없음 시 빈 상태 / 개인정보 마스킹 정책 적용 |
| 관련 화면 | 관리자 화면 |
| 관련 API | `GET /api/admin/members` |

### F-009. SMS 발송

| 항목 | 내용 |
|------|------|
| 설명 | 관리자가 선택 회원에게 SMS를 다이렉트 발송한다 |
| 입력 | 관리자 세션, 수신 대상(회원 목록), 메시지 내용 |
| 처리 | 권한 확인 → 대상 검증 → SMS 사업자 API 호출 → 발송 결과 수집·기록 |
| 출력 | 발송 성공/실패 건수, 결과 목록 |
| 예외 처리 | 사업자 API 오류 시 재시도·실패 기록 / 번호 형식 오류 제외 / 발송 한도 초과 안내 |
| 관련 화면 | 관리자 화면 |
| 관련 API | `POST /api/admin/sms`, 외부 SMS 사업자 API(OI-05) |

### F-011. 로그인·인증

| 항목 | 내용 |
|------|------|
| 설명 | 가족·관리자 로그인 인증을 제공하고 세션을 관리한다 |
| 입력 | 로그인 자격정보(계정/비밀번호 등) |

| 관련 화면 | 가족/관리자 화면 |
| 관련 API | `POST /api/auth/login`, `POST /api/auth/logout` |

### F-012. 권한 분리·접근 제어

|------|------|

| 입력 | 인증 세션의 역할, 요청 리소스 |

| 예외 처리 | 역할 위반 접근 차단·로깅 / 매핑되지 않은 부모님 데이터 접근 차단 |
| 관련 화면 | 전체 |
| 관련 API | 인증 미들웨어(전 API 공통) |

| 예외 처리 | 중복 매핑 방지 / 미존재 계정 매핑 차단 / 매핑 해제 시 접근 즉시 회수 |
| 관련 화면 | 관리자/가족 화면 |
| 관련 API | `GET/POST/DELETE /api/mapping` |

| 관련 화면 | 전체 |
| 관련 API | (해당 없음, 프론트엔드) |

## 예외·공통 처리 정책

| 정책 | 내용 |
|------|------|
| 인증 실패 | 401 반환 + 로그인 유도 |
| 권한 위반 | 403 반환 + 접근 로깅 (F-012) |
| 실시간 연결 끊김 | 자동 재연결 + 누락 이벤트 폴링 보정 (F-005) |
| 외부 API 오류 | 재시도 후 실패 기록, 사용자에 명확한 실패 안내 (F-009/F-010) |
| 위치 권한 거부 | 안내·재요청, 서비스 핵심 사용 제한 고지 (F-001) |
