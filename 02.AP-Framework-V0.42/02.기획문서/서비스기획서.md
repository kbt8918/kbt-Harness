## 1. Executive Summary

## 2. 서비스 배경 및 시장 기회

### 2.1 시장 현황

마켓리서치 결과, 시니어 돌봄 시장은 상위·세부 시장 모두 고성장이며 시니어 위치확인 수요가 실증되었다.

| 항목 | 수치 | 출처 |
|------|------|------|
| 상위 시장 규모 | 글로벌 노인 돌봄 서비스 1.3조$(2024) → 2.1조$(2034), CAGR 5.1% | 마켓리서치 2-8 (NLM-04) |
| 세부 시장 규모 | 글로벌 노인 돌봄 AI 2,085.9억$(2032 전망), CAGR 25.26% | 마켓리서치 2-1 (NLM-02) |
| 타겟 시장 배경 | 국내 1인 가구 804만(36%), 65세+ 1,000만(20.1%), APAC 최고 성장(한국 포함) | 마켓리서치 3.1 (NLM-09, NLM-04) |
| 수요 실증 | 시니어(60세+) 위치앱 가입 +2,136% 급증 | 마켓리서치 2-2 (NLM-09) |
| 시장 성숙도 | 성장기 | 마켓리서치 3.1 |

### 2.2 시장 기회

- **기회 1 — 고성장 시장(이중 검증)**: 노인 돌봄 서비스(CAGR 5.1%, 홈케어 6.2%)와 노인 돌봄 AI(CAGR 25.26%)가 동시 성장, APAC가 최고 성장 지역이고 한국이 대상국에 명시됨 -- 근거: 마켓리서치 4.1-① (NLM-04, NLM-02)
- **기회 2 — 빈 시장(White Space)**: 통신3사·삼성 등 대기업은 하드웨어·관제 고비용 모델에 집중 → 기기 없는 경량 웹 + 가족 직접 연계 영역이 비어 있음 -- 근거: 마켓리서치 4.1-③ (NLM-08, NLM-14)
- **기회 3 — 정책 순풍**: 응급안전안심서비스 대상 확대, 2026 시니어 키워드 'INFRA' → 공공 사각 보완 수요 -- 근거: 마켓리서치 4.1-④ (NLM-12, NLM-13)
- **기회 4 — 검증된 UX 레퍼런스**: 일본 미마모리 "전화 안부=버튼 1회 응답" 패턴이 본 서비스 1-터치 설계를 뒷받침 -- 근거: 마켓리서치 2-5 (NLM-11)

### 2.3 경쟁 환경

## 4. 서비스 전략

### 4.1 핵심 가치 제안 (Value Proposition)

**"감시가 아닌 안심·연결"** — 부모님은 버튼 1회 터치만으로 안부와 위급을 전하고, 가족은 위치·전화·채팅이 통합된 한 화면에서 즉시 대응한다. 별도 기기·설치 없이 반응형 웹으로 누구나 바로 사용한다.

3대 약속:
1. **부모님에겐 단순함** — 큰 버튼·큰 글씨, 1-터치 조작 (마켓리서치 2-3 심리적 장벽 해소)
2. **가족에겐 통합 대응** — 위치+바로전화+가족채팅+긴급알림 한 곳에
3. **모두에게 가벼움** — 기기 불필요, 즉시 사용, 저비용

### 4.2 서비스 범위 (핵심 기능 5종)

| 구분 | 기능 | 우선순위 | 설명 | 대상 화면 |
|------|------|----------|------|-----------|
| MVP | 1. 실시간 위치 공유 | P0 | 부모님 단말 GPS(Geolocation)를 가족 지도에 실시간 표시 | 부모님/가족 |
| MVP | 2. 긴급 알림(1-터치) | P0 | 부모님 버튼 1회 터치 → 가족에게 수 초 내 즉시 알림, 시각적 발송 피드백 | 부모님/가족 |
| MVP | 3. 바로 전화 | P0 | 가족 화면 버튼 클릭 시 부모님에게 즉시 통화 연결(tel:) | 가족 |
| MVP | 4. 가족 채팅 | P0 | 가족 구성원 간 실시간 채팅방으로 정보 공유·대응 조율 | 가족 |
| MVP | 5. 회원 관리·발송 | P0 | 관리자가 연동 회원 조회 + SMS/카카오톡 다이렉트 발송 | 관리자 |
| 확장 | 비활성·낙상 감지 | P1 | 버튼조차 못 누르는 상황 대비 자동 알림 (마켓리서치 2-4 제언) | 차기 로드맵 |
| 확장 | 배터리 잔량 가족 공유 | P1 | 비용·차단 우려 해소(마켓리서치 2-3) | 차기 로드맵 |
| 확장 | 위치 이력·구독 과금 | P2 | 일본 미마모리 구독+출동 모델(마켓리서치 2-5) | 범위 외(향후) |

### 4.3 차별화 전략

1. **단순 조작 기반 긴급 알림**: 부모님 1-터치 설계로 기술 불안·거부감 해소 (PRD FR-P-02/03, NLM-03/NLM-11 검증)
2. **가족 연계 통합**: 단순 위치앱(아이쉐어링) 대비 위치+바로전화+가족채팅을 한 서비스로 통합
3. **"감시 아닌 안심·연결" UX 메시징**: 최대 채택 장벽(감시감) 정면 해소 (마켓리서치 2-3)
4. **경량·저비용 진입**: 대기업 하드웨어·관제 모델이 비운 white space를 반응형 웹으로 공략 (마켓리서치 2-4/2-6)

> 한 줄 표기: `Next.js + Tailwind CSS / Express.js / PostgreSQL / Vercel`

| 영역 | 기술 | 선정 사유 |
|------|------|-----------|
| 프론트엔드 | **Next.js + Tailwind CSS** | SSR/SSG, 반응형, 고령자 친화 UI 빠른 구축 |
