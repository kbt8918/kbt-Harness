# 프롬프트 라이브러리

이 폴더에는 주차별로 사용하는 Claude Code 프롬프트가 정리되어 있습니다.

## 사용 방법

1. 각 주차 파일에서 필요한 프롬프트를 찾습니다
2. Claude Code에서 해당 프롬프트를 입력합니다
3. 생성된 산출물을 검토하고 필요 시 수정을 요청합니다

## 주차별 프롬프트 참조

AP-Framework V0.42의 프롬프트를 참조합니다:
- `AP-Framework-V0.42/prompts/week1-초기화.md` — P-101 ~ P-104
- `AP-Framework-V0.42/prompts/week2-관리자동화.md` — P-201 ~ P-208
- `AP-Framework-V0.42/prompts/week3-기획자동화.md` — P-301 ~ P-309
- `AP-Framework-V0.42/prompts/week4-구현자동화.md` — P-401 ~ P-409
- `AP-Framework-V0.42/prompts/week5-검수자동화.md` — P-501 ~ P-506
